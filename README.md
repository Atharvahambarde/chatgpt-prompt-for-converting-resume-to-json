
# chatgpt-prompt-for-converting-resume-to-json





A ChatGPT Prompt which takes your Resume as input and parses the content in JSON format.
## Prompt:
Sign up for OpenAI. 
Access the prompt portal. 
Enter below text input and input your resume file. 
Watch ChatGPT generate its output.


